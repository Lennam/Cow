TDN计算公式

对于大多数饲料原料的TDN1x：TDN1x(%) = tdNFC + tdCP + (tdFA x 2.25) + tdNDF - 7
动物性蛋白饲料的TDN1x：TDN1x(%) = CPdigest x CP + FA x 2.25 + 0.98 x (100 - CP - Ash - EE) - 7
含甘油的脂肪的TDN1x：TDN1x(%) = (EE x 0.1) + (FAdigest x (EE x 0.9) x 2.25)
不含甘油的脂肪的TDN1x：TDN1x(%) = EE x FAdigest x 2.25
